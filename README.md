# Adopción de mascotas

Sitio web creado como práctica de programación con HTML y CSS. El objetivo es simular una plataforma simple donde los usuarios puedan conocer mascotas disponibles, leer información de la organización y navegar por diferentes secciones del proyecto.

## Descripción

El proyecto reproduce el diseño realizado en Figma y construye una estructura básica de sitio informativo. Cada sección se organizó pensando en navegación sencilla, contenido claro y un estilo visual coherente.
Se utilizaron únicamente HTML y CSS puros; no hay dependencias externas ni frameworks.

## Características principales

* Página de inicio con galería circular para presentar mascotas destacadas.
* Sección "Mascotas" con tarjetas individuales que muestran imagen, nombre y un breve texto.
* Pantalla de detalle para cada mascota (foto ampliada y descripción).
* Página "Nosotros" con información sobre la misión, visión y propósito del refugio.
* Sección "Servicios" con accesos a diferentes opciones del refugio (adopción, educación, donaciones, etc.).
* Diseño responsivo básico inspirado en el prototipo de Figma.
* Uso de degradados, tipografías y jerarquías visuales siguiendo la maqueta original.

## Tecnologías utilizadas

* HTML5
* CSS3
* Figma (como referencia del diseño)

## Cómo ejecutar el proyecto

1. Descarga o clona el repositorio.
2. Abre el archivo "index.html" en cualquier navegador moderno.
3. Navega entre las secciones mediante el menú superior.

No requiere instalación de paquetes ni configuraciones adicionales.

## Objetivos académicos de esta práctica

* Aplicar maquetación HTML semántica.
* Implementar estilos mediante selectores, posicionamiento y uso de cajas.
* Replicar una interfaz desde Figma con precisión razonable.
* Usar gradientes, imágenes, tamaños relativos y otros elementos visuales.
* Organizar el proyecto en carpetas limpias.
